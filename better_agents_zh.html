Better Agents

Better Agents 是一個 CLI 工具，也是一套用於建構 AI 代理（Agent）的標準。

它能為您的程式碼助手（如 Kilocode, Claude Code, Cursor 等）增強能力，使其成為您選擇的任何代理框架（如 Agno, Mastra 等）及其最佳實踐的專家。

這是開始任何新代理專案的最佳方式。

│ └── sample_prompt.yaml ├── prompts.json # 提示詞註冊表 ├── .mcp.json # MCP 伺服器配置 ├── AGENTS.md # 開發指南 ├── .env # 環境變數 └── .gitignore

AGENTS.md 中的結構和指南確保程式碼助手所需的每個新功能都經過適當的測試、評估，並且提示詞受到版本控制。

.mcp.json 附帶了所有正確的 MCP 設置，因此您的程式碼助手將成為您選擇的框架的專家，並知道在哪裡可以找到新工具。

快速開始 (Getting Started)

安裝

npm install -g @langwatch/better-agents

或者使用 npx：

npx @langwatch/better-agents init my-agent-project

初始化新專案

# 在當前目錄初始化
better-agents init .

# 在新目錄初始化
better-agents init my-awesome-agent

CLI 將引導您選擇程式語言、代理框架、程式碼助手、LLM 提供商和 API 金鑰。
